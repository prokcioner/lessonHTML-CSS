PHP and JavaScript

The PHP and JavaScript programming languages allow you to make a website dynamic, that is, resposive to user actions. For examples, you can make a beautiful dropout menu or slider.

Types of scripts

To do this, scripts are written(eng. Script - "сценарий") - programs that allow you to respond to user actions. There are two types of scripts:

  • those that are executed on the server, and the result of their execution will be sent to the browser to the user in a ready-made form. THere are scripts written in PHP, CMS - content managment systems are written on it.
  • those that are executed directly in the user's browser are scripts written in JavaScript. They are most often used to make the page more convenient and beautiful.
